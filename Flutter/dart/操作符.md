# Dart 操作符表

## 算术操作符

| 操作符 | 名称 | 描述 | 示例 |
|-------|------|------|------|
| `+` | 加 | 两个数相加或字符串拼接 | `a + b` 或 `'a' + 'b'` |
| `-` | 减 | 两个数相减 | `a - b` |
| `-` | 负号 | 取数值的相反数 | `-a` |
| `*` | 乘 | 两个数相乘 | `a * b` |
| `/` | 除 | 两个数相除，结果为浮点数 | `10 / 3` // 3.333... |
| `~/` | 整除 | 两个数相除，结果为整数 | `10 ~/ 3` // 3 |
| `%` | 取余 | 两个数相除的余数 | `10 % 3` // 1 |
| `++var` | 前置自增 | 变量值加1，然后返回新值 | `++a` |
| `var++` | 后置自增 | 先返回变量当前值，然后加1 | `a++` |
| `--var` | 前置自减 | 变量值减1，然后返回新值 | `--a` |
| `var--` | 后置自减 | 先返回变量当前值，然后减1 | `a--` |

## 比较操作符

| 操作符 | 名称 | 描述 | 示例 |
|-------|------|------|------|
| `==` | 相等 | 检查两个对象是否相等 | `a == b` |
| `!=` | 不等 | 检查两个对象是否不相等 | `a != b` |
| `>` | 大于 | 检查左边的值是否大于右边 | `a > b` |
| `<` | 小于 | 检查左边的值是否小于右边 | `a < b` |
| `>=` | 大于等于 | 检查左边的值是否大于或等于右边 | `a >= b` |
| `<=` | 小于等于 | 检查左边的值是否小于或等于右边 | `a <= b` |
| `identical()` | 恒等 | 检查两个对象是否是同一个对象 | `identical(a, b)` |

## 逻辑操作符

| 操作符 | 名称 | 描述 | 示例 |
|-------|------|------|------|
| `!` | 非 | 取布尔值的反 | `!a` |
| `&&` | 与 | 逻辑与，短路求值 | `a && b` |
| <code>&#124;&#124;</code> | 或 | 逻辑或，短路求值 | <code>a &#124;&#124; b</code> |

## 赋值操作符

| 操作符 | 名称 | 描述 | 示例 |
|-------|------|------|------|
| `=` | 赋值 | 将右边的值赋给左边的变量 | `a = b` |
| `+=` | 加赋值 | 左边的值加上右边的值，然后赋值给左边 | `a += b` 等价于 `a = a + b` |
| `-=` | 减赋值 | 左边的值减去右边的值，然后赋值给左边 | `a -= b` 等价于 `a = a - b` |
| `*=` | 乘赋值 | 左边的值乘以右边的值，然后赋值给左边 | `a *= b` 等价于 `a = a * b` |
| `/=` | 除赋值 | 左边的值除以右边的值，然后赋值给左边 | `a /= b` 等价于 `a = a / b` |
| `~/=` | 整除赋值 | 左边的值整除右边的值，然后赋值给左边 | `a ~/= b` 等价于 `a = a ~/ b` |
| `%=` | 取余赋值 | 左边的值对右边的值取余，然后赋值给左边 | `a %= b` 等价于 `a = a % b` |
| `<<=` | 左移赋值 | 左边的值左移右边的位数，然后赋值给左边 | `a <<= b` 等价于 `a = a << b` |
| `>>=` | 右移赋值 | 左边的值右移右边的位数，然后赋值给左边 | `a >>= b` 等价于 `a = a >> b` |
| `&=` | 与赋值 | 左边的值和右边的值进行位与操作，然后赋值给左边 | `a &= b` 等价于 `a = a & b` |
| <code>&#124;=</code> | 或赋值 | 左边的值和右边的值进行位或操作，然后赋值给左边 | <code>a &#124;= b</code> 等价于 <code>a = a &#124; b</code> |
| `^=` | 异或赋值 | 左边的值和右边的值进行位异或操作，然后赋值给左边 | `a ^= b` 等价于 `a = a ^ b` |

## 类型测试操作符

| 操作符 | 名称 | 描述 | 示例 |
|-------|------|------|------|
| `as` | 类型转换 | 将对象转换为指定类型 | `obj as String` |
| `is` | 类型检查 | 检查对象是否为指定类型 | `obj is String` |
| `is!` | 非类型检查 | 检查对象是否不是指定类型 | `obj is! String` |

## 位操作符

| 操作符 | 名称 | 描述 | 示例 |
|-------|------|------|------|
| `&` | 位与 | 对两个整数的每一位进行与操作 | `a & b` |
| <code>&#124;</code> | 位或 | 对两个整数的每一位进行或操作 | <code>a &#124; b</code> |
| `^` | 位异或 | 对两个整数的每一位进行异或操作 | `a ^ b` |
| `~` | 位非 | 对整数的每一位进行取反操作 | `~a` |
| `<<` | 左移 | 将整数的位向左移动指定的位数 | `a << b` |
| `>>` | 右移 | 将整数的位向右移动指定的位数 | `a >> b` |

## 条件表达式

| 操作符 | 名称 | 描述 | 示例 |
|-------|------|------|------|
| `condition ? expr1 : expr2` | 三目运算符 | 如果condition为true，返回expr1，否则返回expr2 | `a > b ? a : b` |
| `expr1 ?? expr2` | null-aware 运算符 | 如果expr1不为null，返回expr1，否则返回expr2 | `a ?? b` |

## 级联操作符

| 操作符 | 名称 | 描述 | 示例 |
|-------|------|------|------|
| `..` | 级联操作符 | 允许对同一个对象进行连续的操作 | `obj..method1()..method2()` |
| `?..` | null-aware 级联操作符 | 如果对象不为null，则进行级联操作 | `obj?..method1()..method2()` |

## 其他操作符

| 操作符 | 名称 | 描述 | 示例 |
|-------|------|------|------|
| `()` | 函数调用 | 调用函数或方法 | `func()` 或 `obj.method()` |
| `[]` | 列表访问 | 访问列表或映射中的元素 | `list[index]` 或 `map[key]` |
| `?.` | 条件成员访问 | 如果对象不为null，则访问其成员 | `obj?.method()` 或 `obj?.property` |
| `=>` | 箭头函数 | 用于定义单行函数 | `() => expr` |
| `...` | 展开操作符 | 将可迭代对象的元素插入到另一个集合中 | `[...list]` |
| `...?` | null-aware 展开操作符 | 仅当可迭代对象不为null时展开 | `[...?list]` |
