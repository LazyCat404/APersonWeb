# Dart 类与面向对象编程

## 声明类

> 在 Dart 中，类是面向对象编程的基本构建块，用于定义对象的属性和行为。通过 `class` 关键字声明类。

```dart
// 定义一个类（大驼峰）
class Person {
  // 类的属性
  String name;
  int age;

  // 类的方法
  void sayHello() {
    print('Hello, my name is $name and I am $age years old.');
  }
}

// 创建一个类的实例（对象）
Person person = Person();
// Person person = new Person(); // 也可以使用 new 关键字创建实例
person.name = 'Alice';
person.age = 30;
person.sayHello(); // 输出: Hello, my name is Alice and I am 30 years old.
```

## 静态成员、私有成员

* 静态成员：使用 `static` 关键字声明的成员，属于类而不是实例。可以通过类名直接访问，无需创建实例。

* 私有成员：使用下划线 `_` 前缀声明的成员，只能在库（文件）内部访问，不能从库外部访问。

* Dart 没有 `public`、`protected`、`private` 等访问修饰符，所有成员默认都是 public。

```dart
class Person {
  String name;
  static int age; // 静态

  // 构造函数
  Person(this.name);
  
  // 静态方法
  static void sayInfo(Person person) { // 通过参数接收实例
    print("Name: ${person.name}"); // 通过实例参数访问实例属性
    print("age: $age"); // 静态属性
  }
}

void main() {
  Person alice = Person('Alice');
  Person.sayInfo(alice); // 传递实例参数
}

```
